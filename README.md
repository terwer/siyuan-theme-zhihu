## Recommand IDE

vscode

## Useful scripts

### Develop css

```bash
pnpm watch:css
```

### Develop js

```bash
pnpm dev
```

### Build

```bash
pnpm ci
```

### Publish to npm

```bash
pnpm package
```

### Docs

```bash
pnpm vitepress:dev
```
